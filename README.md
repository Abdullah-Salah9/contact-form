# Contact Form with Validation

A responsive contact form built using HTML, CSS, and JavaScript.

## Description
This project is a clean and responsive contact form designed to work well on both desktop and mobile devices.

The form includes the following fields:
- Full Name
- Email Address
- Subject
- Message
- Submit Button

## Features
- Responsive layout that adapts to different screen sizes
- Clean UI with proper spacing between inputs
- Basic form validation using JavaScript
- Required fields validation
- Email format validation

## Technologies Used
- HTML Forms
- CSS Styling
- Responsive Design
- JavaScript (DOM manipulation & validation)
